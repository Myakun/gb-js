Задача 1

Дан код
var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3
Почему код даёт именно такие результаты?

Решение задачи 1

Код
c = ++a;
alert(c);
Выведет 2
Процитирую методичку: "В префиксной форме инкрементирование производится сразу, а возврат — уже с обновленным значением".
Код
d = b++;
alert(d);
Выведет 1
Процитирую методичку: "В постфиксной форме сначала происходит возвращение значения, а потом выполняется инкрементирование".
Код
c = (2+ ++a);
alert(c);
Выведет 5
Переменная "a" имела значение 2.
Вновь процитирую методичку: "В префиксной форме инкрементирование производится сразу, а возврат — уже с обновленным значением".
В итоге мы складывали 2 и 3, что равно 5.
Код
d = (2+ b++);
alert(d);
Выведет 4
Переменная "a" имела значение 2.
Вновь процитирую методичку: "В постфиксной форме сначала происходит возвращение значения, а потом выполняется инкрементирование".
В итоге мы складывали 2 и 2, что равно 4.
Код
alert(a);
alert(b);
Выведет 3 для обеих переменных. Их значения были равны 1, в ходе работы программы каждая переменная была инкрементирована по два раза.

Задача 2

Чему будет равен x в примере ниже?
var a = 2;
var x = 1 + (a *= 2);

Решение задачи 2

x равен 5.
Выражение *= выполняет умножение на 2, при этом переменная "a" станет равна 4.
