Задача 4

Самостоятельно разобраться с атрибутами тега script (async и defer).

Решение задачи 4

async - асинхронная загрузка скрипта, которая не тормозит отображение страницы.
defer - общий смысл аналогичен async, но сохраняет последовательность загрузки по порядку.
